# FocusTimer — Smart Pomodoro + Break Games 

A modern productivity web app that combines the Pomodoro technique with interactive mini-games during breaks to keep you refreshed and motivated.

# Features
 Smart Pomodoro Timer :
* Work, Short Break, Long Break modes
* Customizable durations
* Auto-start options

# Productivity Tracking
* Daily Pomodoro count
* Focus time tracking
* Streak system

## Task Manager
* Add, complete, and delete tasks
* Track Pomodoros per task

# UI & Experience
* Dark / Light mode
* Multiple gradient themes
* Clean and responsive design

# Smart Notifications
* Alarm sound when session ends
* Browser notifications
* Motivational quotes when skipping tasks

# Break Games (Unique Feature)

Play games during breaks:
* Snake 🐍
* Tic Tac Toe ❌
* Memory Match 🃏
* Speed Typing ⌨️
* 2048 🔢
* Flappy Bird 🐦
* Word Guess 📝
* Breakout 🧱

Break ends automatically and brings you back to focus mode.

# Tech Stack
* HTML5
* CSS3 (Custom Properties, Responsive UI)
* JavaScript (Vanilla JS)
* Web APIs (LocalStorage, Notifications, Audio)

📁 FocusTimer
│── index.html      # UI structure

# How to Run
1. Download or clone the repository
2. Open the folder in VS Code
3. Open `index.html` in your browser

OR simply double-click the file.

# How It Works

* Start a Pomodoro session
* Focus on your task
* After session ends → take a break
* Play mini-games during break
* Automatically return to work mode
